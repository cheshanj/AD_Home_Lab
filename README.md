# Active Directory Home Lab

This repository contains a complete **step-by-step guide** to setting up an **Active Directory environment** using **VirtualBox**, **Windows Server 2022**, and **Windows 10**.  
The full detailed guide is available in [`docs/Active_Directory_Home_Lab.md`](docs/Active_Directory_Home_Lab.md).

## Overview
- Simulate a real-world Active Directory environment
- Configure Domain Controller, DHCP, DNS, NAT, and internal networking
- Join Windows 10 clients to the domain and verify connectivity

## Technologies Used
- Oracle VirtualBox
- Windows Server 2022
- Windows 10 Pro
- Active Directory Domain Services (AD DS)
- DHCP, DNS, NAT, RAS
